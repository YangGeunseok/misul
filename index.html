작가노트

작업 주제: 일상에서 발견되는 조형적 질서를 사진으로 기록하고 재구성하는 작업이다.

제작 과정: 조명 방향과 간결한 도형을 반복하여 형태의 경계와 그림자의 관계를 실험했다.

감상 포인트: 각 이미지의 구도에서 시선의 흐름을 따라가 보세요 — 빛이 만드는 음영과 여백의 관계를 주목하면 좋습니다.

템플릿 B — 스토리 흐름형

추천: 사진마다 단계적 서사가 있을 때

출발점 — 영감의 순간

확장 — 기법 실험

서사 텍스트 예시

프롤로그: 이 시리즈는 '관찰'에서 시작되었다. 작은 일상에서 반복되는 형상들이 눈에 들어왔고, 이를 사진으로 기록해 구조화하려는 시도이다.

중간: 빛을 바꾸고, 배경을 단순화하며 형태 간의 관계를 계속해서 재편집했다.

에필로그: 완성된 이미지들은 한 가지 정서(침묵, 정돈)를 공유하며 관람자에게 여운을 남기도록 구성되었다.

템플릿 D — 연구보고서형

추천: 심화된 텍스트와 분석이 필요할 때

요약

본 연구는 일상적 대상의 형상성을 사진으로 분석하고, 빛과 각이 결합되었을 때 발생하는 시각적 의미를 탐구한다.

분석 (사진별)

  2. 사진 2: 형태 분절의 효과
  3. 사진 3–4: 실험 과정과 수정 기록
  4. 사진 5: 최종적 해석
